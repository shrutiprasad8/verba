**ğŸš€ Getting Started**

To run this project on your local machine, please follow the steps below.

**Prerequisites**
Node.js (v18 or later recommended)

npm (comes with Node.js)

**Local Setup & Configuration**

*Clone the repository*:

git clone [https://github.com/shrutiprasad8/verba.git](https://github.com/shrutiprasad8/verba.git)
cd verba

*Install dependencies*:

npm install

*Add Your API Key*:
This project requires a Google Gemini API key to generate summaries.

Get a free API key from Google AI Studio.

Open the src/App.jsx file.

Find the apiKey variable and replace the placeholder string ("YOUR_GEMINI_API_KEY_HERE") with your actual key.

*Run the development server*:

npm run dev

The application will be available at http://localhost:5173.
